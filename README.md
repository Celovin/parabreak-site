# Parabreak — parabreak.com

Marketing site for Parabreak. Next.js 16 App Router, deployed to Vercel.

Current state: pre-launch placeholder. Phase 2 design (logo lockup, full landing
mockup, OG/favicon assets) is still in flight with Claude Designer. Once those
land, the 11-section landing layout from `parabreak/landing/` in the engine
sources gets ported in here.

## Develop

```bash
npm install
npm run dev
```

## Deploy

Vercel production only. Preview deployments are disabled by policy. The site is
expected at https://parabreak.com via Cloudflare DNS pointing at Vercel.
